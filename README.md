# CampusConnect

CampusConnect is a university event management web system built using HTML, CSS, JavaScript, PHP, and MySQL.

## 📌 Project Description
The system allows students to:
- View university events
- Register for events
- Contact event organizers

It also includes an admin panel that allows administrators to:
- Create and manage events
- Edit and delete events
- Manage user registrations
- Store and retrieve data using a MySQL database

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## 🎯 Features
- Student registration system
- Event browsing
- Admin dashboard
- Database integration
